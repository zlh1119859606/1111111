# 个人网站项目

## 项目简介
这是一个极简风格的个人网站项目，采用纯 HTML/CSS/JavaScript 实现，无需复杂的框架和构建工具。

## 项目目标
创建一个具有苹果审美风格的极简个人网站，展示个人作品和创意表达。

## 技术栈
- HTML5
- CSS3（原生CSS，无框架）
- JavaScript（原生JS，无库依赖）

## 项目结构
```
个站/
├── index.html          # 主页面
├── css/
│   └── style.css      # 样式文件
├── js/
│   └── main.js        # 交互脚本
└── README.md          # 项目说明
```
